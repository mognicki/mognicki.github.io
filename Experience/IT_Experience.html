Chosen projects

A GIS-driven CRM

For a Warsaw-based company, specializing in the remediation of lake waters, I’ve built a simple Laravel-based CRM, enhanced with advanced GIS functionalities.

The core objective of the application was to retrieve GIS data on Polish lake waters, including their quality, and discover potential marketing possibilities. Data sources included:

  • Polish Main Sanitary Inspectorate,
  • Shapefiles retrieved from Nominatim and Polish Cartographic Service.

Base data were processed using QGIS, and imported into the application as GeoJSON files. Dynamically imported data on the lake waters quality were then regularly processed to assess marketing possibilities.

E-mail redistribution system

I’ve built a simple, Laravel-based e-mail redistribution application, used for assigning marketiing leads to company’s partners. The application was integrated with Yahoo mail service (client’s requirements) and employed a custom algorithm for partner’s random drawing (enhanced with partners’ priority/weight flags).

Dietary application

Using Laravel and it’s queuing features, I’ve provided a complex dietary application, integrated with Nexmo for interaction with clients via text messages.

The application used advanced logic for dietary notifications, and multiple case scenarios for each user.

Nuxt.js website for Polish distributor of the WIWE device

For the Polish distributor of the WIWE device - a mobile EEG apparatus - I’ve created a SPA website, using Nuxt.js and underlying Express.js server.

Website has been integrated with a number of third-party services, including Polish REGON service, MongoDB Atlas, a Tpay.com payment gate, as well as the Mailgun, and a number of SEO services.
